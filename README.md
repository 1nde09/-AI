# -AI
‘모의면접 AI 인터뷰’는 면접 준비와 대학 지원 전략을 하나의 플랫폼에서 해결하는 통합 입시 도우미 서비스입니다.  사용자는 대학·진로별 맞춤 면접 질문을 선택하고, 음성 인식 기반으로 실제 면접처럼 답변할 수 있습니다. AI는 답변의 논리 구조·표현력·발음·긴장도 등을 분석하여 정확하고 구체적인 피드백을 제공합니다. 이를 통해 혼자 연습하기 어려운 실전 감각과 말하기 능력을 효과적으로 향상시킬 수 있습니다.  또한, 사용자는 자신의 수능 성적(백분위/표준점수)을 입력하면, 입력한 성적을 기반으로 지원 가능한 대학과 합격 가능성이 높은 추천 대학을 확인할 수 있습니다.
